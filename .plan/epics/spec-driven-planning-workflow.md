# Spec-Driven Planning Workflow

Created: 2026-04-16T05:33:06Z

## Outcome

Ship the core planning workflow that makes `plan` genuinely useful: brainstorm entry, epic creation, spec approval, story creation, and project status.

## Why Now

Without the workflow, `plan` is just a folder generator. This epic turns the workspace into an actual planning tool.

## Scope Boundary

- `plan brainstorm ...`
- `plan epic ...`
- `plan spec ...`
- `plan story ...`
- `plan status`
- spec approval gate before stories

Not in scope:

- dependency graph
- automated plan checking
- roadmap editing power features
